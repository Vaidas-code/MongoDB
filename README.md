# MongoDB
MongoDB yra duomenų bazė, kuri duomenis saugo kaip JSON tipo dokumentus (BSON formatas), leidžianti lengviau valdyti struktūrizuotus ir nestruktūrizuotus duomenis.
MongoDB palaiko įdėtus dokumentus ir masyvus, leidžiančius natūraliai modeliuoti sudėtingas duomenų struktūras.
Naudojama galinga ir lanksti užklausų kalba su filter, aggregation ir index palaikymu.
Duomenys gali būti dubliuojami (replication) į kelis mazgus, užtikrinant duomenų prieinamumą ir atsparumą gedimams.
Tinka aplikacijoms, kuriose reikia lankstumo, realaus laiko duomenų analitikos, didelio duomenų kiekio valdymo ir greito vystymo.
